# Hellox Code of Conduct

## Our commitment

We want Hellox to be a respectful, practical, and inclusive place for contributors,
users, and researchers working on local-first coding-agent tooling.

Participation in this project means you agree to help keep discussions constructive,
kind, and focused on the work.

## Expected behavior

- Be respectful in language and tone.
- Assume good intent and ask for clarification before escalating.
- Share technical feedback with concrete examples.
- Keep disagreements focused on design, behavior, and evidence.
- Welcome contributors with different backgrounds and experience levels.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Threats, intimidation, or sustained hostile behavior.
- Publishing private information without permission.
- Deliberate disruption, trolling, or bad-faith engagement.
- Sexualized language or unwanted attention.

## Scope

This code of conduct applies to:

- repository issues, pull requests, discussions, and reviews
- project chat or coordination spaces related to Hellox
- public interactions representing the project

## Reporting

If you experience or observe unacceptable behavior:

- do not escalate publicly if the report includes sensitive personal or security details
- contact the repository maintainer through GitHub
- include links, screenshots, and a brief timeline when possible

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull
requests, and other contributions that violate this code of conduct.

Maintainers may also temporarily or permanently restrict participation for repeated
or severe violations.
